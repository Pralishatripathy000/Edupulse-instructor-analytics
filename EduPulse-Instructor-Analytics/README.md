# 📊 EduPulse

### AI-Powered Instructor Effectiveness Analytics

EduPulse analyzes instructor performance using educational engagement metrics and machine learning models.

## Tech Stack

Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn, XGBoost, Jupyter Notebook

## Project Structure

- `data/raw` - original dataset
- `data/processed` - cleaned data
- `notebooks` - analysis notebook
- `models` - saved models
- `outputs` - charts, reports, predictions
